# Customer Purchase Analytics

**Scalable Customer Purchase Analytics Platform** is a modular, memory-efficient Python package designed to process large datasets (10M+ rows) in chunks.

## Key Features

- **Memory Efficiency**: Processes data in chunks to handle datasets larger than memory.
- **Reproducibility**: metrics and segmentation logic are versioned and testable.
- **Modular Design**: Structured as a reusable Python package.

## Tech Stack

- **Language**: Python
- **Libraries**: Pandas, Numpy
