# La Liga Analytics

**La Liga Analytics** is a data consolidation pipeline designed to fetch, merge, and analyze football data from multiple sources.

## Key Features

- **Data Consolidation**: Merges data from APIs and CSVs.
- **Analytics**: Generates comprehensive stats on players, teams, and matches.
- **Visualization**: Interactive dashboards for trend tracking.

## Tech Stack

- **Pipeline**: Python, Pandas, Apache Airflow
- **Storage**: Azure Data Lake Storage Gen2 (DLS2)
- **Visualization**: Power BI, Matplotlib
